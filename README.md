# RPG Dungeon Crawler

**Version 1.0.0** - Work in Progress

A text-based RPG dungeon crawler adventure game built with .NET 10.

## Features

- Character creation with multiple races and classes
- Dungeon exploration and combat system
- Quest and achievement system
- Pet companions
- Crafting and trading
- Multiplayer support
- Skill trees and leveling
- Equipment and inventory management
- Dynamic weather and time of day
- Bank and gambling systems

## Development Status

⚠️ **This is a work-in-progress project**

## Requirements

- .NET 10.0 Runtime or higher
- Windows x64

## Build

```bash
dotnet build -c Release
```

## Publish

```bash
dotnet publish -c Release -r win-x64 --self-contained
```
