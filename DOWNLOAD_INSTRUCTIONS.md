# 🎮 Download Instructions for v2.0.0

## ⚠️ IMPORTANT: Windows Defender Warning

**Windows will flag this game as potentially harmful - THIS IS A FALSE POSITIVE!**

The game is completely safe and open-source. Windows flags it because:
- It's not digitally signed (code signing costs $300+/year)
- It's a new release without download history
- Self-contained .NET apps trigger heuristic detection

### ✅ How to Run the Game:

**When Windows SmartScreen appears:**
1. Click "**More info**"
2. Click "**Run anyway**"
3. Done! The game will start

**Detailed instructions**: See `WINDOWS_DEFENDER_HELP.md` inside the zip file

## 🚀 Installation Steps

1. **Download** your preferred zip file
2. **Extract** to any folder (e.g., `C:\Games\RPG_Dungeon\`)
3. **Right-click** `ConsoleApplication.exe` → Properties → Check "**Unblock**" (if present) → Apply
4. **Run** `ConsoleApplication.exe`
5. When Windows SmartScreen appears: Click "**More info**" → "**Run anyway**"
6. **Enjoy!**
