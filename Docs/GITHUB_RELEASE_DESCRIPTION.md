## ⚠️ WINDOWS SECURITY WARNING - READ FIRST!

**Windows will show a security warning when you run this game.**

### This is a FALSE POSITIVE - The game is 100% SAFE! ✅

**Why does this happen?**
- The game isn't "digitally signed" (costs $300+/year for indie developers)
- Windows doesn't recognize new apps without signatures

### 🎯 How to Play:
1. Download the zip file below
2. Extract it
3. Run `ConsoleApplication.exe`
4. When Windows shows the warning:
   - Click "**More info**"
   - Click "**Run anyway**"
5. Enjoy! 🎮

## 🎉 WHAT'S NEW IN VERSION 2.0.0

This is a **MASSIVE UPDATE** with tons of new content!

### 🏆 12 Champion Classes
Unlock powerful specializations at level 20:
- **Warriors**: Berserker, Paladin, Guardian, Templar
- **Rogues**: Assassin, Shadowblade, Ranger
- **Mages**: Archmage, Elementalist, Necromancer
- **Special**: Druid, Oracle

Each with unique abilities, passives, and talents!

### 🗺️ Explore a Vast World
- Travel between major towns, settlements, and camps
- Discover hidden locations (fog of war system)
- Random encounters during travel
- Multiple biomes and terrain types

### ⚒️ Master 6 Crafting Professions
- **Blacksmithing** - Forge weapons and heavy armor
- **Leatherworking** - Craft light armor and accessories
- **Alchemy** - Brew potions and elixirs
- **Enchanting** - Imbue items with magic
- **Jewelcrafting** - Create rings and amulets
- **Tailoring** - Weave cloth armor

Plus: Recipe learning, resource gathering, quality tiers!

### 🏋️ Train Your Champion
- New **Training Hall** to learn abilities
- Unlock special talents
- Practice combat rotations

### ⚔️ Enhanced Combat
- Tactical stances (Aggressive/Defensive/Balanced)
- Status effects (Bleeding, Stunned, Poisoned, Burning, Frozen, and more)
- Dynamic encounters

### 🎯 More Features
- Quest and achievement systems
- Pet companions
- Banking and trading
- Day/night cycle and weather
- And much more!
